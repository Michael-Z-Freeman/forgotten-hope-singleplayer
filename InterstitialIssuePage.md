## Forgotten Hope Single Player Forum ##

The [Forgotten Hope single players forum](http://fhpubforum.warumdarum.de/index.php?board=63.0) is for discussion of issues related to the playing of the single player and coop games. Anything related to FHSP is appropriate with topics including help on the usage of a feature, bugs, and new features you would like to see.

## List Issues ##

[List of current issues](http://code.google.com/p/forgotten-hope-singleplayer/issues/list).

## Report an Issue ##

If you have found an issue, please report it using the issue tracker. Before you report an issue, browse through the [list of current issues](http://code.google.com/p/forgotten-hope-singleplayer/issues/list) and see if it has already been reported. If it has not, [submit a new issue](http://code.google.com/p/forgotten-hope-singleplayer/issues/entry) and give a detailed description of the problem. Provide screenshot attachments if necessary but please keep file size as small as possible. One of the Forgotten Hope single player developers or a member of the single player community will take a look at it and get back to you.
